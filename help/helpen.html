Introduction

This tool is dedicated for manage entries of AUCOTEC.INI file. Managed entries of licence and colour set. Additionaly will possible to edit ELCAD_ID file.

Tab Licence

In this tab you can select method of licencing flying or hardlock. In case flying licence you can select product (eg. ELCAD, AUCOPLAN etc.), interfaces (e.g. DXF, PDF, etc.) and moduls (e.g. STUDIO, HCL etc.).

First you have to select file to edit. Is possible change directory. You don't have to select exact user directory, tool search all AUCOTEC.INI in subdirectories and lists they together.

Tab Colors

In this tab you can select colour set.

For example: if you select set version 35, you will get black background

Tab Editor

In this tab you can edit lecence file ELCAD_ID.
